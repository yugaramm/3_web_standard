▷ 자바스크립트
	정적 웹 : HTML, CSS 파일의 코드가 직접 수정되기 전까진 항상 같은 내용을 표시
	동적 웹 : 이미 해석이 끝난 웹 브라우저에 표시된 요소의 내용을 인위적으로 조작.

	HTML과 CSS는 정적 웹만을 구현할 수 있다.
	이를 동적 웹처럼 동작하게 하려면 자바스크립트 언어를 사용해야한다.

▶ 자바스크립트 역할
	1) 사용자와의 상호작용 및 연산
		폼은 입력을 할 수 있는 공간만 제공할 뿐, 계산 및 처리는 오직 자바스크립트로만 가능.
	2) 웹 페이지 내에 HTML, CSS 동적 제어
	3) 웹 서버와의 통신
		서버와 데이터를 주고 받을 수 있다.

▶ 자바스크립트 코드
	웹 브라우저 -> 개발자 모드 -> console 탭에서 단순한 자바스크립트 문법을 이용한 코드 작성 및 실행 가능.
	(에러가 났다면, console 탭에서 확인 가능!)	

	VS Code 확장 프로그램
	Code Runner -> VS Code 에서 별도의 웹 브라우저 없이 코드 실행 및 확인 가능.
	node.js -> 공식 홈페이지 들어가서 다운로드. 한글 깨짐 없이 코드를 읽어서 반환해줌.

▶ 변수
	키워드 식별자 = 값; 

	키워드
		var : 변수를 선언할 때 사용하는 키워드. 자료형은 존재하지만 변수를 만들 때 지정할 필요는 없다. (함수 레벨 스코프를 가짐)
		let : var 이 후에 나온 변수 선언 키워드. 중복 생성 불가능, 호이스팅이 되지 않는다. (블록 레벨 스코프를 가짐)
		const : 상수 선언 시 사용.

▶ 특수 자료형
	undefined : 변수 공간에 값을 할당하지 않을 경우, 자바스크립트가 임시로 데이터를 할당하는 값. 개발자가 정의하는 것이 아님.
	null : 선언 시 의도적으로 공간을 비워둘 때 개발자 할당.

▶ 객체
	기본 자료형 빼고는 전부 자바스크립트 내의 객체라고 보면 된다.
	크게 배열, 객체 리터럴, 함수 객체로 나눌 수 있다.
