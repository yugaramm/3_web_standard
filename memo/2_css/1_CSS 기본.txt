▷ CSS 란?
	웹 구조에 스타일을 적용하기 위해 작성하는 시트

▶ 기본 문법
	div {
		color : red; <-- 세미콜론으로 줄을 구분할 수 있다.
		font-size : 49px;
	}

▶ CSS 를 적용하는 3가지 방법
	1. 인라인 스타일 적용
		HTML의 style 속성을 사용하여 CSS 코드를 작성하는 방법.
		<태그 style="CSS 코드">내용</태그>

	2. 내부 스타일 시트
		HTML 문서 내에 style 태그를 활용하여 CSS 코드를 추가, 적용할 수 있다.	

	3. 외부 스타일 시트
		.css 확장자를 가진 별도의 파일을 만들어 CSS 코드를 작성한 후, HTML 문서와 연결하는 방식이다.
		HTML 문서 내에 스타일 시트를 등록하여 사용할 수 있다.

		<link rel="stylesheet" href="css 파일 경로">
