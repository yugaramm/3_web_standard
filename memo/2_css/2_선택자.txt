▷ 선택자
	선택자란 스타일을 지정할 요소를 특정하는 문법이다.
	여러가지 방식이 존재한다.

▶ 기본 선택자
	전체 선택자 : *
	태그 선택자 : 태그 이름
	아이디 선택자 : #속성값 -> id 속성값을 이용하여 스타일을 지정. 하나의 HTML 문서 안에서 고유한 값이어야 한다.
	클래스 선택자 : .속성값 -> id와 다르게 중복해서 사용할 수 있어, 한 번에 여러 곳에 지정 후 한 번에 스타일을 먹일 수 있다.
	속성 선택자 : 선택자[속성] : 태그의 해당 속성이 있는 요소 선택
	 	   선택자[속성=값] : 태그의 해당 속성이 있고, 값도 일치하는 요소 선택

▶ 조합 선택자
	그룹 선택자 : 선택자, 선택자, 선택자, ...
	자식 선택자 : 부모 선택자 > 자식 선택자
	하위 선택자 : 선택자 선택자 선택자 -> 자식에만 한정이 아닌 손주 중손주 까지 전부 적용
	형제 선택자1 : 선택자 + 선택자 -> 첫번째 선택자 다음으로 오는 형제 요소에 적용
	형제 선택자2 : 선택자 ~ 선택자 -> 첫번째 선택자와 형제인 요소들에게 모두 적용

▶ 가상 요소 선택자
	HTML 문서에 실제 작성된 요소는 아니지만, 존재하는 것처럼 취급해 선택하는 지정 방법을 말한다.
	쉽게 말해 선택한 요소의 일부에만 스타일을 적용하는 것인데, 새로운 요소를 추가한 것처럼 동작한다.
	실무에서 많이 사용되는 두 가지만 알아보자!

	선택자::before { } , 선택자::after { }

	각각 해당 선택자의 앞의 공간, 뒤의 공간을 선택할 수 있다.
	content 속성을 활용하여 새로운 텍스트를 추가할 수 있고, 해당 텍스트에 스타일도 먹일 수 있다.

▶ 가상 클래스 선택자
	요소의 상태를 이용하여 선택자로 지정하는 방식.

	선택자:가상 클래스 선택자 { }

	1) 동적 - 어떤 행동에 따라 변하는 상태에 따라 선택자를 지정할 수 있다.
		- :active : 요소를 마우스로 클릭하고 있는 동안 선택자로 지정
		- :hover : 마우스를 올릴 때 선택자로 지정

	2) a 태그 : 링크의 방문 여부에 따라 선택자를 지정할 수 있다.
		- :link : 방문하지 않은 링크일 때 선택자로 지정
		- :visited : 방문한 적 있는 링크일 때 선택자로 지정

	3) input 태그 : 상호 작용 요소의 상태에 따라 선택자를 지정할 수 있다.
		- :focus : 요소가 선택될 때 선택자로 지정
		- :checked : 체크박스가 체크되어있을 경우 선택자로 지정
		- :disabled : disabled 속성이 부여되어 있을 때 선택자로 지정
		- :enabled : disabled 가 아닐 때 선택자로 지정

	4) 구조적 가상 클래스
	
▶ CSS 적용 우선 순위
	일단 태그들마다 고유하게 부여 되어있는 스타일이 존재한다.
	스타일을 어디든 어떻게 선택해서 먹이든 고유하게 부여되어있는 스타일은 사라진다고 보면 된다.

	우선순위가 같다면 마지막으로 읽힌 스타일이 적용된다. (단계적 적용)

	- 선택자별 우선 순위 정리
	인라인 스타일 	1000
	아이디		100
	가상 클래스		10
	클래스		10
	가상 요소		1
	태그		1
	전체		0

	굳이 이 점수를 디테일하게 외우고 점수 계산할 필요 없이, 자식 태그로 내려가는 것 하위 요소로 찾아가는 것 등등으로
	선택자를 선택한다면 더 우선순위가 높다. 정도의 느낌만 가져도 충분.
