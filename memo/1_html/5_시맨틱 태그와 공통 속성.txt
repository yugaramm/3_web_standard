▷ 시맨틱 태그 (sementic, 의미적인)
	웹 페이지의 구조를 설계하기 위한 목적으로 등장했다.

▶ header 태그
	웹 페이지의 헤더 영역을 구분하기 위해 사용한다.

▶ nav 태그
	다른 링크로 이동해주는 주요 탐색 링크 영역을 포함한다.

▶ section 태그
	논리적인 내용 영역을 구분하는 용도로 사용하면 좋다.

▶ article 태그	
	독립적인 영역을 구분할 때 사용하면 좋다.

▶ aside 태그	
	웹 페이지와 관련 없는 내용을 영역으로 묶을 때 사용한다.

▶ footer 태그	
	최하단 영역을 구분할 때 사용한다.


▷ 공통 속성
	▶ class	
		여러 요소들이 중복으로 가질 수 있는 선택자를 지정할 수 있다.

	▶ id
		중복이 될 수 없는 선택자를 지정할 수 있다.

	▶ style
		CSS 코드를 인라인으로 작성할 때 사용할 수 있다.

	▶ title
		해당 요소에 마우스를 올릴 때 나오는 툴팁을 지정할 수 있다.

	▶ hidden
		요소를 화면에서 감출 때 사용한다.
