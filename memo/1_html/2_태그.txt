▷ div, span 태그
	코드를 작성하다 보면 관련 있는 요소를 그룹으로 묶어야하는 경우가 빈번하게 생긴다.
	그룹으로 묶어주면 레이아웃을 구성하기가 쉬워지고, 구조를 깔끔하게 정리, 작성할 수 있다.
	그룹으로 묶어주는 행위는 다른 영역과 분리한다는 의미로도 해석이되며,
	div, span 태그를 공간 분할 태그라고도 부른다.

1. div 태그 -> 블록 요소 (division)
	블록 요소와 인라인 요소를 그룹으로 묶을 때 사용한다.
	그룹으로 묶을 때 사용하는 대표적인 블록 요소 태그이다.

2. span 태그 -> 인라인 요소
	인라인 요소를 그룹으로 묶을 때 사용한다.
	텍스트 일부분에만 스타일을 입히고 싶을 때 주로 사용한다.
	
▷ 링크, 이미지 태그
	링크는 문서와 문서간의 연결을 의미하고, a 태그를 사용한다.
	사진도 넣을 수 있는데 img 태그를 사용한다.
	
	<a href="경로" target="연결 방식" title="설명"></a>
	target : _blank, _parent, _self, _top 있지만, _blank 만 사용
	title : 설명을 작성, 둘 다 생략 가능.

	<img src="이미지 경로" alt="이미지 설명">

▷ 목록 관련 태그
	목록은 목차나 메뉴를 구성할 때 주로 사용하지만, 다양한 곳에서 사용 가능하다.

1. ul 태그
	순서가 없는 목록을 생성한다. 목록 내용은 li 태그이다.

2. ol 태그
	순서가 있는 목록을 생성한다. 목록 내용은 li 태그이다.

3. dl 태그
	정의형 목록을 생성한다.
	내용 중 단어나 용어는 dt 태그, 설명은 dd 태그를 이용한다.
