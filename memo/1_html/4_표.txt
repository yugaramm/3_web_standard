▷ HTML 로 웹페이지에 표 그리기

▶ table 태그
	폼 구성하기의 form 태그와 마찬가지로 표와 관련된 태그는 table 태그 내부에 작성한다.

▶ caption 태그
	표에 제목을 작성하는 태그

▶ tr 태그 (table row)
	표에 행을 생성할 때 사용하는 태그

▶ th, td 태그 (table head, data)
	실제 데이터를 생성하는 영역으로, 열을 만들어주면서 데이터를 생성한다.

▶ rowspan, colspan 속성
	행과 열을 병합하는 것이라고 생각하면 된다.

▶ thead, tbody, tfoot 태그 속성
	태그의 이름 그대로 테이블의 제일 윗부분과 아랫부분을 지정할 때 사용할 수 있는 태그이다.
	활용하지 않아도 상관은 없지만 접근성 향상을 위해 사용하는 것을 권장한다.

▶ col, colgroup 태그
	열을 그룹화 하기 위해 사용한다.
	열마다 그룹으로 묶어 스타일을 지정하거나 할 때 유용하게 사용할 수 있다.

▶ scope 태그
	데이터를 적어주는 곳에 데이터가 아닌, 제목을 넣어줄 때 사용한다.
	웹 접근성 향상을 목적으로 사용하는 경우가 많다.
